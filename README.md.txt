# Library Management System (Java + JDBC + MySQL)

A simple menu-driven Library Management System.

## Features
- View Books
- Add Book
- View Students
- Issue Book

## How to Run
1. Import the database from `database.sql`.
2. Open `LibraryOperations.java` in any IDE (IntelliJ/VS Code).
3. Run the program and use the menu.

## Technologies
- Java
- JDBC
- MySQL / MariaDB
